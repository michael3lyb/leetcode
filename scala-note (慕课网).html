  1. 函数式编程重要概念
    1. 纯函数(pure Function), 或函数的纯粹性(purity)没有副作用(side effect)
      1. 副作用是状态的变化
        1. 修改全局变量
        2. 抛出异常
        3. IO读写
        4. 调用有副作用的函数
          1. 改变变量(var)的值
    2. 引用透明(referential transparency)  对相同的输入，总是可以得到相同的输出
    3. 如果f(x)的参数x和函数体都是引用透明的，那么函数f是纯函数
    4. 不变性(Immutability) 为了获得引用透明性，任何值都不能变化
    5. 函数是第一等公民 (first-class Function)
      1. 一切都是计算，函数式编程中只有表达式，变量、函数都是表达式
    6. 高阶函数(Higher order Funtion)
      1. 一个函数可以作为另一个函数的输入或输出
    7. 闭包(Clouse)
    8. 表达式求值策略(Evaluation Strategy)
      1. 严格求值 Call By Value - 对函数实参求值，且仅求值一次
        1. 通常使用
        2. def foo(x: Int) = x
        3. test1(x: Int, y: Int): Int = x * x
          执行过程
          test1(3+4, 8)
               test1(7,8)
                    7*7
                     49
      2. 非严格求值 Call By Name - 函数实参每次在函数体内被用到时都会求值
        1. 以=> 开头
        2. def foo(x: => Int) = x  
        3. test1(x: => Int, y: => Int): Int = x * x
          执行过程
          test1(3+4, 8)
                (3+4)*(3+4)
                    7*(3+4)
                      7*7
                       49
      3. 惰性求值(Lazy Evaluation)
  2. 函数式编程优点
    1. 生产效率高
    2. 易于推理(Reasoning)
    3. 并行编程，多核计算，云计算

  4. 柯里化: programming in scala 9.3
    1. 柯里化函数(Curried Function)把具有多个参数的函数转换为一条函数链，每个节点上是单一参数。
    2. 例子：以下两个函数定义是等价的
      def add(x: Int, y:Int)  = x + y
      def add(x: Int)(y: Int) = x + y //scala中柯里化的语法
    3. 例子
def curried Add(a: Int)(b: Int) = a + b
curriedAdd(2)(2) //4
val addOne = curriedAdd(1)_  //Int=>Int
addOne(2)  //3
  1. 递归函数（Recursive Function ）
    1. 函数式编程没有循环
    2. 必须指定返回类型
    3. 函数式编程中是实现循环的一种技术
    4. 例子:计算n!
      def factorial(n: Int): Int = 
        if (n <= 0) 1 
        else n * factorial(n - 1)
    5. 尾递归(tail Recursion): 用于调优递归，防止堆栈溢出
      1. 所有递归形式的调用都出现在函数的末尾
      2. 当编译器检测到一个函数调用是尾递归的时候，它就覆盖当前的活动记录而不是在栈中去创建一个新的。
      3. 例子：
        @annotation.tailrec
        def factorial(n: Int, m: Int): Int =
          if (n <= 0) m
          else factorial(n - 1, m * n)

        factorial(5, 1)   //m*n!

  def qSort(a : List [Int]) : List [Int] = {
    if (a. length < 2) a
    else qSort(a .filter( a.head > _)) ++
      a.filter (a. head == _) ++    //或者List(a.head)
